# Fake Loading Button

A simple desktop app idea prototype:
- Big red `do not press` button
- Fake loading bar that runs for 20 seconds
- Plays a random sound at completion

## Run

From repo root:

```powershell
python apps/fake_loading_button/fake_loading_button.py
```

## Sound Source

This app reuses `.wav` files from `apps/button/sounds/`.
